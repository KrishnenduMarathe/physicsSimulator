## CONTROLS

#### KEYS       ||    ACTION
#### --------------------------------------------
#####  W         =>    Move UP
#####  A         =>    Move Left
#####  S         =>    Move Down
#####  D         =>    Move Right
#####  `         =>    Exit Simulation
#####  ?         =>    Display Debug Window
#####  |         =>    Start Object Rotation (constant speed)
